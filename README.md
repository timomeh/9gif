#9gif

> Download gifs from 9gag. Powered By Giphy.

## Usage

```sh
$ 9gif <9gag-url> <destination (optional, defaults to cwd)>

# Example
$ 9gif http://9gag.com/gag/amz4Y9X ~/gifs
```
